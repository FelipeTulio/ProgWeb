Tag XHTML: label

Exemplo

Um formulário HTML simples com dois campos de entrada e etiquetas relacionadas:
<form>
<label for="male"> Masculino </ label>
<input type="radio" name="sex" id="male" />
<br /> <label for="female"> </ label> Feminino
<input type="radio" name="sex" id="female" />
</ form><br/>




Feminino

Definição e uso

A tag <label> define um rótulo para um elemento de entrada.
O elemento label não torna como algo especial para o usuário. No entanto, ele fornece uma melhoria de usabilidade para os usuários do mouse, pois se o usuário clicar sobre o texto dentro do elemento label, que alterna o controle.
O atributo for da tag [label] deve ser igual ao atributo id do elemento relacionado com os mantêm juntos.

Atributos opcionais

Atributo Valor Descrição DTD
for element_id Specifies which form element a label is bound to STF

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset
