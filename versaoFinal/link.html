Tag XHTML: link

Exemplo

Link para uma folha de estilo externo:
<head>
<link rel="stylesheet" type="text/css" href="theme.css" />
</ head>

Definição e uso

A tag <link> define a relação entre um documento e um recurso externo.
A tag [link] é mais usado para ligações para folhas de estilo.
