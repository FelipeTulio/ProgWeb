Definição e uso

A tag XHTML a define valores de atributos para uma ou mais colunas em uma tabela.

A tag col é útil para aplicação de estilos de colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.

A tag col só pode ser usado dentro de uma tabela ou um elemento colgroup.
