Tag XHTML: table

Exemplo

<table border="1">
<tr>
<th> Mês </ th>
Poupança <th> </ th>
</ Tr>
<tr>
<td> janeiro </ td>
<td> $ 100 </ td>
</ Tr>
</ Table>

Definição e uso

A tag XHTML table define uma tabela HTML.
A tabela HTML simples consiste no elemento da tabela e um ou mais tr, th, e os elementos td.
O elemento tr define uma linha da tabela, o elemento º define um cabeçalho da tabela, eo elemento td define uma célula da tabela.
A tabela mais complexa HTML também pode incluir legenda, col, colgroup, thead, tfoot, tbody e elementos.
