Tag XHTML: option

Exemplo

A lista suspensa com quatro opções:

<select> <option>Volvo</option>
<option>Saab</option>
<option>Mercedes</option>
<option>Audi</option>
</select>

Definição e uso

A tag <option> define uma opção em uma lista de seleção.
O elemento de opção vai dentro do elemento select.

Atributos opcionais

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
disabled disabled
right
top
bottom		 Especifica que a opção deve ser desativada. STF
label text Especifica um curto rótulo para uma opção. STF
selected selected Especifica que a opção deve ser selecionada por padrão. STF
value text Especifica o valor a ser enviado para um servidor quando um formulário for enviado. STF

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
