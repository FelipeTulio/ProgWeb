Definição e uso

A tag XHTML tbody é usada para agrupar o conteúdo do corpo em uma tabela HTML.
O elemento tbody deve ser usado em conjunto com o thead e tfoot.
O elemento thead é usado para agrupar o conteúdo do cabeçalho em uma tabela HTML eo elemento tfoot é usado para agrupar o conteúdo do rodapé em uma tabela HTML.
Nota: tfoot deve aparecer antes tbody dentro de uma tabela, de modo que um navegador pode tornar a pé antes de receber todas as linhas de dados.
Nota-se que estes elementos não afetaram o layout da tabela por padrão. No entanto, você pode usar CSS para deixar esses elementos afetem o layout da tabela.
