Tag XHTML: script

Exemplo

Inserir um JavaScript em uma página HTML:

<scrip type = "text/javascript">
document.write ("Olá Mundo!")
</scrip>

document.write ("Olá Mundo!")

Definição e uso

A tag <script> é usado para definir um script do lado do cliente, tais como JavaScript.
O elemento de script ou contém declarações de script ou ele aponta para um arquivo de script externo através do atributo src.
O atributo tipo exigido especifica o tipo MIME do script.Usos comuns para JavaScript são de manipulação de imagem, validação de formulário, e alterações dinâmicas do conteúdo.

Atributos opcionais

Atributo Valor Descrição DTD
charset charset Specifies the character encoding used in an external script file STF
defer defer Specifies that the execution of a script should be deferred (delayed) until after the page has been loaded STF
src URL Specifies the URL of an external script file STF
xml:space preserve Specifies whether whitespace in code should be preserved  

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

The <script> tag does not support any standard attributes.
