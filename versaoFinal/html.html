Tag XHTML: html

Exemplo

Um documento HTML simples, com o mínimo de tags necessário:
<html><br/>
<head><br/>
<title> Título do documento </ title><br/>
</ head><br/>
<body><br/>
O conteúdo do documento ...... <br/>
</ body><br/>
</ html><br/>;

Definição e uso

A tag <html> informa ao navegador que este é um documento HTML.
O elemento html é o elemento mais externo em documentos HTML e XHTML.
O elemento html é também conhecido como o elemento raiz.

Atributos Necessários

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
xmlns http://www.w3.org/1999/xhtml Especifica o namespace a ser usado (somente para documentos XHTML!) STF

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
dir rtl
ltr		 Especifica a direção do texto para o conteúdo de um elemento STF
lang language_code
ltr		 Especifica um código de linguagem para o conteúdo de um elemento STF
xml:lang language_code Especifica um código de linguagem para o conteúdo de um elemento, em documentos XHTML STF
