Tag XHTML: sub

Exemplo

<p> This contains <sub>subscript</sub> text.</p>
<p> This contains <sup>superscript</sup> text.</p>

This contains subscript text.

This contains superscript text.

Definição e uso

A tag <sub> define o texto subscrito. Texto subscrito aparece meia um personagem
abaixo da linha de base. Texto subscrito pode ser usado para fórmulas químicas,
como H2O.
A tag <sup> define texto sobrescrito. Texto sobrescrito meia aparece um personagem
acima da linha de base. Texto sobrescrito pode ser utilizada para notas de
rodapé, como WWW [1].
