Tag XHTML: address

Exemplo

<address> Written by W3Schools.com
<a href="mailto:us@example.org">Email us</a>
Address: Box 564, Disneyland
Phone: +12 34 56 78
</address>

Written by W3Schools.com
Email us
Address: Box 564, Disneyland
Phone: +12 34 56 78

Definição e uso

A tag XHTML address define as informações de contato para o autor ou titular de um documento. Desta forma, o leitor é capaz de contatar o dono do documento.

O elemento address normalmente é adicionado ao cabeçalho ou rodapé de uma página web.
