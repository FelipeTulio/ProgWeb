Tag XHTML: var

Exemplo

Variable <var> Variable </ var>

Definição e uso

A tag XHTML var é uma tag frase. Ela não está obsoleta, mas é possível conseguir mais rico efeito com CSS.

Descrição da Tag
<var> Define uma variavel para o texto

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
class nome de classe Especifica o nome da classe para o elemento STF
dir rtl
ltr		 Especifica a direção do texto para o conteúdo de um elemento STF
id identificação Especifica uma identificação única para o elemento STF
lang language_code Especifica um código de linguagem para o conteúdo de um elemento STF
style definição de estilo Especifica um estilo em linha STF
title texto Especifica uma informação extra sobre o elemento STF
xml:language language_code Especifica um código de linguagem para o conteúdo de um elemento, em documentos XHTML STF
