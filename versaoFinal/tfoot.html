XHTML Tag Tfoot

A tag [tfoot] é usado para agrupar o conteúdo do rodapé em uma tabela HTML.
O elemento tfoot deve ser usado em conjunto com o thead tbody e elementos.
O elemento thead é usado para agrupar o conteúdo do cabeçalho em uma tabela HTML eo elemento tbody é usado para
agrupar o conteúdo do corpo em uma tabela HTML.
Nota: [tfoot] deve aparecer antes [tbody] dentro de uma tabela, de modo que um navegador pode tornar a pé antes de receber todas as linhas de dados.Note-se que estes elementos não afetará o layout da tabela por padrão. No entanto, você pode usar CSS para deixar esses elementos afetam o layout da tabela.
