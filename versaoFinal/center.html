Tag XHTML: center

Exemplo

<center> Este texto será centralizado. </ center>

Definição e uso

A tag XHTML center é usada para centralizar o texto.

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
class nome de classe Especifica o nome da classe para o elemento TF
dir rtl
ltr		 Especifica a direção do texto para o conteúdo de um elemento TF
id identificação Especifica uma identificação única para o elemento TF
lang language_code Especifica um código de linguagem para o conteúdo de um elemento TF
style definição de estilo Especifica um estilo em linha TF
title texto Especifica uma informação extra sobre o elemento TF
