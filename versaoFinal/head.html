Tag XHTML: head

Exemplo

Um documento HTML simples, com o mínimo de tags necessário:
<html>
<head>
<title>Title of the document</title>
</head>
<body>
The content of the document......
</body>
</html>

Definição e uso

A tag XHTML O elemento principal é um recipiente para todos os elementos da cabeça.
Elementos dentro [head] pode incluir scripts, instruir o browser onde encontrar folhas de estilo, meta fornecer informações e muito mais.
As seguintes tags podem ser adicionadas à seção head: [base], [link], [meta], [script], [style] e [title].
A tag [title] define o título do documento, e é o único elemento exigido na seção head!

Atributos opcionais

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
profile url Especifica um URL para um documento que contém um conjunto de regras.
As regras podem ser lidos por navegadores para entender claramente as informações em atribuir a tag [meta] de conteúdo.		 STF
