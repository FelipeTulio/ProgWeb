Tag XHTML: span

Definição e uso

A tag <span> é usado para agrupar inline-elementos em um documento.A tag <span> não fornece nenhuma mudança visual por si só.
A tag <span> fornece uma maneira de adicionar um gancho para uma parte de um texto ou uma parte de um documento.
Quando o texto é viciado em um elemento span que você pode adicionar estilos ao conteúdo, ou manipular o conteúdo com que o JavaScript exemplo.

Exemplo

Um texto com um elemento span que podem ser decorados com CSS:
<p> Minha mãe tem <span class="blue"> luz azul </ span> olhos. </ p>
