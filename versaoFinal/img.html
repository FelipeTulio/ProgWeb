Tag XHTML: img

Definição e uso

A tag XHTML A tag <img> incorpora uma imagem em uma página HTML.
Observe que as imagens não são tecnicamente inserido em uma página HTML, as imagens são links para páginas HTML. A tag <img> cria um espaço para manter a imagem de referência.
A tag <img> tem dois atributos necessários: src e alt.

Exemplo

Uma imagem é marcada como segue:
<img src="angry.gif" alt="Angry face" /><br/>
