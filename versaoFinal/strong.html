Tag XHTML: strong, em, dfn, code, samp, kbd, var, cite

Exemplo

Formatar texto em um documento:
<strong>Strong text</strong>
<em>Emphasized text</em>
<dfn>Definition term</dfn>
<code>Computer code text</code>
<samp>Sample computer</samp>
<kbd>Keyboard text</kbd>
<var>Variable</var>
<cite>Citation</cite>

Definição e uso

O <em>, <strong>, <DFN>, <code>, <samp>, <kbd>, <var> e tags <cite> são todas as
tags frase. Eles não estão obsoletos, mas é possível conseguir mais rico efeito.
