Definição e uso

A tag <pre> define texto pré-formatado.
Texto em um elemento pré é exibido em uma fonte de largura fixa (geralmente Courier), e preserva os espaços e quebras de linha.

Atributos necesários

DTD: S = Strict; T = Transitional; F = Frameset

Atributos opcionais

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
width number Obsoleta. Use estilos css em vez de XHTML4.
Especifica o número máximo de caracteres por linha.		 TF
