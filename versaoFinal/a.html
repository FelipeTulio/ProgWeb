Tag XHTML: a

Exemplo

<a href="http://www.ifnmg.edu.br">Site do IFNMG</a>

Definição e uso

A tag XHTML a define uma âncora. Uma âncora pode ser usada de duas formas:

  1. para criar uma ligação com outro documento pela definição do atributo href;
  2. para criar um marcador no próprio documento pelo uso do atributo id.

O elemento a é, frequentemente, chamdo de link ou hyperlink.

Por padrão, links aparecem sublinhados e nas seguintes formas:

unvisited
não visitado e em azul.
visited
visitado e em roxo.
active
ativo e em vermelho.

Atributos opcionais

Atributo Valor Descrição DTD
href URL Espefica o destino da ligação STF
target _blank
_parent
_self
_top		 Especifica onde o documento da ligação será aberto STF

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
accesskey caractere Espcifica um atalho de teclado para acessar o elemento STF
class nome de classe Especifica o nome da classe para o elemento STF
id identificação Especifica uma identificação única para o elemento STF
style definição de estilo Especifica um estilo em linha STF
tabindex número Especifica a ordem de tabulação do elemento STF
title texto Especifica uma informação extra sobre o elemento STF
