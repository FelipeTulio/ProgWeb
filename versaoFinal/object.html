Tag XHTML: object

Exemplo

Adicionar um objeto (arquivo Flash) para uma página HTML:
<object width="400" height="400" data="helloworld.swf"> </ object>

Definição e uso

A marca <object> é usado para incluir objetos como imagens, áudio, vídeos, applets
Java, ActiveX, PDF e Flash.
O elemento de objeto se destinava a substituir os elementos img e applet. No entanto,
por causa de erros e falta de suporte ao navegador, isso não aconteceu.
O suporte objeto em browsers dependem do tipo de objeto. Infelizmente, os principais
navegadores usam códigos diferentes para carregar o mesmo tipo de
objeto.
Felizmente, o elemento de objeto fornece uma solução. Se o elemento de objeto não
for exibida, o código entre as <object> e </object> será executado. Desta forma,
podemos ter vários elementos objeto aninhado (um para cada navegador).
