Tag XHTML: q

Exemplo

Uma citação curta é marcada como segue:
<q> Aqui está uma citação curta aqui é uma citação curta
</q> Aqui está uma citação curta aqui é uma citação curta

Definição e uso

A tag define uma cotação curta. O browser irá inserir aspas em torno da cotação

Atributos opcionais

Atributo Valor Descrição DTD
cite URL Specifies the source of a quotation STF
