XHTML Tag Select

Exemplo

<select>
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="mercedes">Mercedes</option>
<option value="audi">Audi</option>
</select>

Definição e uso

A tag [select] é usado para criar uma lista de seleção (drop-down list).
As tags [option] dentro do elemento select definir as opções disponíveis na lista.

Atributos opcionais

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
disabled disabled Especifica que uma lista drop-down deve ser desativado. STF
mutiple mutiple Especifica que múltiplas opções podem ser selecionadas. STF
name name Especifica o nome de uma lista suspensa. STF
size number Especifica o número de opções visíveis em uma lista suspensa. STF

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
class classname
ltr		 Especifica um nome de classe para um elemento STF
dir rtl
ltr		 Especifica a direção do texto para o conteúdo de um elemento STF
id id Especifica um id único para um elemento STF
lang language_code
ltr		 Especifica um código de linguagem para o conteúdo de um elemento STF
style style_definition
ltr		 Especifica um estilo interno para um elemento STF
tabindex number
ltr		 Especifica a ordem de tabulação de um elemento STF
title text
ltr		 Especifica informações adicionais sobre um elemento STF
xml:lang language_code Especifica um código de linguagem para o conteúdo de um elemento, em documentos XHTML STF
