Tag XHTML: button

Exemplo

<button type="button"> Clique em Mim! </ button>

Definição e uso

A tag button define um botão.

Dentro de um elemento botão, você pode colocar o conteúdo, como texto ou imagens.
Esta é a diferença entre este elemento e botões criados com o elemento de entrada.
Sempre especificar o atributo type para o botão. O tipo padrão para Internet Explorer é o "botão", enquanto que em outros navegadores (e na especificação W3C) é "submit".

Atributos opcionais

Atributo Valor Descrição DTD
desabled desativado Especifica que um botão deve ser desativado STF
name nome Especifica o nome de um botão STF
type tipo Especifica o tipo de um botão STF
value texto Especifica o valor sobjacente de um botão STF
