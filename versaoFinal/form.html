Definição e uso

A tag é usado para criar um formulário HTML para entrada do usuário.
Um formulário pode conter elementos de entrada, como campos de texto, checkboxes,
botões de rádio, botões de envio e muito mais. Uma forma também pode conter
menus select, textarea, fieldset, legend, e elementos do rótulo.
Os formulários são usados para transmitir dados para um servidor.
