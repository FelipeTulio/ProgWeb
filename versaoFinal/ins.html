Um texto com uma parte excluída e uma nova peça inserida:

Minha cor favorita é azul vermelho

Definição e uso

A tag <ins> define o texto que foi inserido em um documento.

Atributos opcionais

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
cite url Especifica um URL para um documento que explica por que o texto foi inserido / alterado STF
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data ea hora em que o texto foi inserido ou alterado. STF

Atributos padrão

DTD: S = Strict; T = Transitional; F = Frameset

Atributo Valor Descrição DTD
class classname Especifica um nome de classe para um elemento STF
dir rtl
ltr		 Especifica a direção do texto para o conteúdo de um elemento STF
id id
ltr		 Especifica um id único para um elemento STF
lang language_code
ltr		 Especifica um código de linguagem para o conteúdo de um elemento STF
style style_definition
ltr		 Especifica um estilo interno para um elemento STF
title text
ltr		 Especifica informações adicionais sobre um elemento STF
xml:lang language_code Especifica um código de linguagem para o conteúdo de um elemento, em documentos XHTML STF
