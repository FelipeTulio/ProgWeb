Tag XHTML: style

Exemplo

Uso do elemento de estilo em um documento HTML: <html>
<head>
<style type="text/css">
h1 {color:red;}
p {color:blue;}
</style>
</head>
</html>

Definição e uso

A tag <style> é usado para definir informações de estilo para um documento HTML.
Dentro do elemento de estilo que você especificar como os elementos HTML deve processar em um navegador.
O atributo tipo exigido define o conteúdo do elemento de estilo. O único valor possível é "text / css".
O elemento de estilo sempre vai dentro da seção head.
